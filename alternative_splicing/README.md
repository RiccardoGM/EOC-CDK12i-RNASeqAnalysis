## Alternative Splicing (AS) analysis

The downstream alternative splicing (AS) analysis is performed using the AS_analysis.py script.
This script processes the output generated by rMATS and produces summary tables and figures describing significant splicing events.

### Requirements

Python version 3.9.20  

Libraries:  
- numpy 2.0.2  
- pandas 2.2.3  
- matplotlib 3.9.2   

### Folder Structure

- rmats_output/  
  The subdirectory structure must follow: LINENAME/Condition1vsCondition2  
  Where: LINENAME refers to the biological model or dataset being analyzed. Condition1vsCondition2 corresponds to the comparison performed by rMATS. NB: Condition1 and Condition2 must match the sample groups specified in the rMATS execution command (--b1 and --b2 BAM file lists). Each comparison folder must contain the standard rMATS output files:  

  A3SS.MATS.JCEC.txt  
  A3SS.MATS.JC.txt  
  A5SS.MATS.JCEC.txt  
  A5SS.MATS.JC.txt  
  MXE.MATS.JCEC.txt  
  MXE.MATS.JC.txt  
  RI.MATS.JCEC.txt  
  RI.MATS.JC.txt  
  SE.MATS.JCEC.txt  
  SE.MATS.JC.txt  

  These correspond to the five rMATS splicing event types: skipped Exon (SE), Retained Intron (RI), Mutually Exclusive Exons (MXE), Alternative 5' Splice Site (A5SS), and Alternative 3' Splice Site (A3SS).

- scripts/  
  This folder includes the files below:  

  AS_analysis.py – main analysis script  
  utils.py – helper functions used during processing  
  parameters.json – configuration file defining analyses settings and file paths  
  samples_loop.sh – script to loop through predefined cell lines and comparison condition pairs 

- figures/  
  Empty folder - will store all generated figures.   


### How to Run

Set up (parameters.json):

- AS parameters:  
  only_junctions - true/false implies that .JC.txt/.JCEC.txt files are used  
  FDR_threshold - maximum False Discovery Rate used to identify significant splicing events  
  abs_delta_psi_threshold - minimum absolute difference in Percent Spliced In required to consider an event biologically relevant  
  avg_reads_threshold - minimum average read support required across samples for a splicing event to be retained  

- Paths:  
  rmats_output_dir - directory containing all rMATS outputs organized by line and comparison  
  figures_dir - output directory where all generated plots will be saved  
  gtf_file - genome annotation file used for mapping events to genes  
  gene_info_file - additional gene annotation metadata used during summarization  

Once the parameters are set, execute a single analysis with  

python AS_analysis.py --line_name "$linename" --cond1 "$Condition1" --cond2 "$Condition2"  

or loop through different cell lines and comparisons with  

./samples_loop.sh
